## 📖 LibreChat이란?

**LibreChat**은 ChatGPT의 오픈소스 대안입니다. 여러 AI 모델을 하나의 인터페이스에서 사용할 수 있는 웹 기반 채팅 플랫폼입니다.

### 핵심 개념
- **오픈소스**: 무료로 사용하고 수정할 수 있음
- **멀티 AI 지원**: OpenAI, Claude, Google 등 여러 AI를 한 곳에서 사용
- **자체 호스팅**: 자신의 서버에서 실행하여 데이터를 완전히 제어
- **ChatGPT 스타일 UI**: 친숙한 인터페이스

---

## ✨ 주요 기능 요약

### 1. 🤖 AI 모델 선택
**무엇을 하는가?** 여러 AI 서비스를 한 곳에서 사용

**지원하는 AI 서비스:**
- **상용 서비스**: OpenAI (ChatGPT), Anthropic (Claude), Google (Gemini), Azure OpenAI
- **로컬 AI**: Ollama (컴퓨터에서 직접 실행)
- **기타**: Groq, Mistral AI, Cohere, OpenRouter 등

**왜 유용한가?**
- ChatGPT Plus 구독 없이도 여러 AI를 사용 가능
- 각 AI의 장점을 상황에 맞게 선택 가능
- 무료 API를 활용하여 비용 절감

---

### 2. 🔧 코드 인터프리터 (Code Interpreter)
**무엇을 하는가?** 채팅에서 직접 코드를 실행하고 파일을 처리

**지원 언어:**
- Python, JavaScript/TypeScript, Go, C/C++, Java, PHP, Rust, Fortran

**기능:**
- 파일 업로드 및 처리
- 코드 실행 결과 확인
- 안전한 샌드박스 환경에서 실행

**사용 예시:**
- 데이터 분석 (CSV 파일 처리)
- 이미지 처리
- 수학 계산
- 코드 테스트

---

### 3. 🔦 에이전트 & 도구 통합
**무엇을 하는가?** AI가 특정 작업을 수행하도록 도구를 연결

**에이전트 기능:**
- **코드 없는 커스텀 어시스턴트**: 특정 작업에 특화된 AI 도우미 생성
- **에이전트 마켓플레이스**: 다른 사람이 만든 에이전트 사용
- **공유 기능**: 특정 사용자/그룹과 에이전트 공유

**도구 예시:**
- 파일 검색
- 코드 실행
- 웹 검색
- 데이터베이스 조회

---

### 4. 🔍 웹 검색
**무엇을 하는가?** AI가 인터넷을 검색하여 최신 정보를 가져옴

**작동 방식:**
1. 사용자가 질문
2. AI가 관련 검색어로 인터넷 검색
3. 검색 결과를 분석하여 답변 제공

**장점:**
- 최신 정보 제공
- 사실 확인 가능
- 다양한 소스 참조

---

### 5. 🪄 생성형 UI (Generative UI)
**무엇을 하는가?** 채팅에서 직접 React 컴포넌트, HTML, 다이어그램 생성

**예시:**
- 대화형 차트 생성
- HTML 페이지 미리보기
- Mermaid 다이어그램 그리기

---

### 6. 🎨 이미지 생성 및 편집
**무엇을 하는가?** 텍스트로 이미지를 생성하거나 기존 이미지를 편집

**지원 서비스:**
- DALL-E 2/3 (OpenAI)
- Stable Diffusion (로컬)
- Flux
- GPT-Image-1

**기능:**
- 텍스트 → 이미지
- 이미지 → 이미지 (스타일 변경, 편집)

---

### 7. 💾 프리셋 & 컨텍스트 관리
**무엇을 하는가?** 자주 사용하는 AI 설정을 저장하고 재사용

**기능:**
- **프리셋 저장**: 특정 AI 모델과 설정 조합 저장
- **대화 중 전환**: 대화 중에 다른 AI 모델로 전환 가능
- **메시지 편집**: 이전 메시지 수정 및 재전송
- **대화 분기**: 대화를 여러 갈래로 나누어 탐색

---

### 8. 💬 멀티모달 & 파일 상호작용
**무엇을 하는가?** 이미지와 파일을 업로드하여 AI와 대화

**지원 기능:**
- **이미지 분석**: Claude 3, GPT-4o 등으로 이미지 내용 설명
- **파일 채팅**: PDF, 텍스트 파일 등을 업로드하여 질문

**사용 예시:**
- 스크린샷 분석
- 문서 요약
- 코드 파일 설명

---

### 9. 🌎 다국어 지원
**무엇을 하는가?** 인터페이스를 여러 언어로 사용

**지원 언어 (20개 이상):**
- 영어, 한국어, 일본어, 중국어(간체/번체)
- 스페인어, 프랑스어, 독일어, 이탈리아어
- 러시아어, 아랍어, 히브리어 등

---

### 10. 🧠 추론 UI
**무엇을 하는가?** AI의 사고 과정을 단계별로 보여줌

**특징:**
- Chain-of-Thought (사고의 사슬) 모델 지원
- DeepSeek-R1 같은 추론 모델에 최적화

---

### 11. 🗣️ 음성 & 오디오
**무엇을 하는가?** 음성으로 대화하고 음성으로 답변 받기

**기능:**
- **음성 입력**: 마이크로 말하기
- **음성 출력**: AI 답변을 음성으로 듣기
- **자동 전송/재생**: 음성 입력 후 자동 전송, 답변 자동 재생

---

### 12. 📥 대화 가져오기/내보내기
**무엇을 하는가?** 다른 플랫폼의 대화를 가져오거나 LibreChat 대화를 내보내기

**지원 형식:**
- LibreChat 형식
- ChatGPT 형식
- Chatbot UI 형식

**내보내기 형식:**
- 스크린샷
- Markdown
- 텍스트
- JSON

---

### 13. 🔍 검색 & 발견
**무엇을 하는가?** 모든 메시지와 대화를 검색

**기능:**
- 전체 대화 검색
- 메시지 내용 검색
- 빠른 검색 결과

---

### 14. 👥 멀티 유저 & 보안 접근
**무엇을 하는가?** 여러 사용자가 안전하게 사용

**인증 방법:**
- 이메일/비밀번호
- OAuth2 (Google, GitHub, Discord 등)
- LDAP (기업용)
- SAML (기업용)

**기능:**
- 사용자 관리
- 권한 설정
- 토큰 사용량 추적
- 콘텐츠 필터링

## 🎯 LibreChat을 사용해야 하는 이유

### 1. 비용 절감
- ChatGPT Plus 구독 불필요
- 무료 API 활용 가능
- 사용량에 따른 유연한 비용

### 2. 프라이버시
- 자신의 서버에서 실행
- 데이터 완전 제어
- 외부 서비스와 데이터 공유 없음

### 3. 유연성
- 여러 AI 모델 선택
- 커스터마이징 가능
- 확장 가능한 구조

### 4. 오픈소스
- 무료 사용
- 커뮤니티 지원
- 코드 수정 가능

- ✅ 자체 호스팅 가능
